# 🎭 Are the Arts Dying? Data Analysis Project

## 📊 Project Overview

This project conducts a comprehensive analysis of the state of performing arts in Canada, with a focus on opera and musical theatre organizations. It investigates whether these sectors are experiencing measurable decline by examining key indicators such as attendance levels, number of performances, and financial performance over time.

Using national data from 2014 to 2024, the project involves cleaning and restructuring raw datasets, building a relational database to organize the information, and applying both SQL queries and Python based analysis to uncover trends and relationships. The analysis includes year over year comparisons, correlation between attendance and revenue, and visualization of patterns to better understand how audience engagement and financial stability have evolved.

The goal is to evaluate a common cultural claim that traditional performing arts are fading.


## 🧠 Research Question

Are opera and musical theatre companies in Canada experiencing declines in attendance, performances, or revenue over time?

## 📁 Data Source

The dataset is sourced from Statistics Canada and includes national level data on performing arts organizations.

It contains information on attendance, performances, and multiple revenue streams for not for profit companies.

## 🧩 Methodology

### Data Processing

Raw datasets were cleaned and transformed into structured formats suitable for analysis. This included filtering by industry classification, reshaping data, and handling missing values.

### Database Design

A relational SQLite database was created to store cleaned data. Tables were designed for attendance and revenue, enabling efficient querying and analysis.

### Data Analysis

SQL queries and Python functions were used to:
Analyze trends over time
Compute year over year changes
Measure correlation between attendance and revenue

### Visualization

Line charts and comparative plots were created to clearly illustrate trends and relationships across variables.
